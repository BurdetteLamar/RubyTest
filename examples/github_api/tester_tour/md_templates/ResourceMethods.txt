# Resource Methods

The data objects in this framework offer methods for accessing the API resources:

- CRUD methods:
  - `create`
  - `read`
  - `upate`
  - `delete`
- Getter methods:
  - `self.get_all`
  - `self.get_first`
- Existence methods:
  - `exist?`
  - `verdict_assert_exist?`
  - `verdict_refute_exist?`
  - `delete_if_exist?`
